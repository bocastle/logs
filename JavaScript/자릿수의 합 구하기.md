# 💡 JavaScript - 자릿수의 합 구하기

## 📝 문제 설명
자연수 `N`이 주어졌을 때, `N`의 **각 자릿수의 합**을 구해서 반환하는 함수 `solution`을 작성하세요.

예를 들어, `N = 123`이면 `1 + 2 + 3 = 6`을 반환해야 합니다.

---

## 📌 제한 사항
- `N`은 **100,000,000 이하의 자연수**입니다.

---

## ✅ 입출력 예

| N     | return |
|-------|--------|
| 123   | 6      |
| 987   | 24     |
| 7     | 7      |
| 1000  | 1      |
| 99999 | 45     |

---

## 🔍 예시 설명

### ✔ 예제 1
**N = 123**  
각 자릿수: 1, 2, 3  
→ 총합: `1 + 2 + 3 = 6`

### ✔ 예제 2
**N = 987**  
각 자릿수: 9, 8, 7  
→ 총합: `9 + 8 + 7 = 24`

### ✔ 예제 3
**N = 1000**  
각 자릿수: 1, 0, 0, 0  
→ 총합: `1`

### ✔ 예제 4
**N = 99999**  
각 자릿수: 9, 9, 9, 9, 9  
→ 총합: `9 × 5 = 45`

---

## 💻 코드 구현 (JavaScript)

```javascript
function solution(n) {
  return n.toString()
          .split('')
          .map(Number)
          .reduce((sum, digit) => sum + digit, 0);
}
